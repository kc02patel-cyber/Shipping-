# Flexport-Style Logistics Intelligence Dashboard

A world-class logistics intelligence system built for investor-grade demonstrations, built on top of a multimodal 5,000-shipment dataset.

## Summary

This system provides:
- Trade lane flow analytics (Sankey)
- Congestion heatmaps
- Carbon intensity scoring
- Carrier performance benchmarking
- Live streaming simulation
- Automated intelligence insights

Designed for:
- Supply chain investors
- Freight forwarders
- Climate-tech funds
- Trade-tech startups
- VC demo days
- Command center PoCs

## Dataset

Included fields:
- Mode / Carrier / Container Type
- Lead Time / Distance / Weight
- Carbon Emissions
- Cargo Value
- Seller-Receiver Corridor
